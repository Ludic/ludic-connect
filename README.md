# Ludic-Connect

#### A Ludic.js extension - WebRTC based networking, for multiplayer ####


## Installing

#### jspm

`jspm install lc=github:Ludic/ludic-connect`

#### webpack

`npm install github:Ludic/ludic-connect`

## Testing
`npm run test`

## Using
```javascript
import lc from 'ludic-connect'

lc.createLobby('testLobby')
lc.findLobbies();

```
